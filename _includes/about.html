About the Project

The Rahway River Project was created by Tejas Gupta to raise awareness about the Rahway River and its conservation efforts. Learn about the problems affecting the Rahway River and how you can help .

This siteis its main tool to inform local communities about how they can help preserve our natural resource, the Rahway River. It was coded by Tejas Gupta, and the source code is available at the project's Github Repository .

Please share this website on Facebook, Twitter, and other social networks to raise awareness and help the cause.
